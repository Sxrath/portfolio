Data Encryption System

"Secure Banking System with MySQL Integration" ensures secure financial transactions by employing the `cryptography.fernet` library for robust encryption. When users register or perform transactions, their account balances are encrypted using unique keys derived from user-defined 4-digit PINs. This encryption guarantees data privacy, making financial information unreadable even if the database is compromised. This comprehensive approach ensures a trustworthy and private banking experience."

Employee Management System

This Python project, named "Employee Data Management System," provides functionalities for HR personnel and employees. HR staff can insert, view, update, and delete employee records, while employees can view their own records. The program uses a CSV file for data storage, enabling seamless management of employee details. The HR module allows HR staff to insert new records, view all records, update existing data, and delete records. Employees can view their own data. This system simplifies employee data management for organizations, ensuring data accuracy and accessibility.
